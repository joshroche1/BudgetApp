Budget App

Python web application built on FastAPI and Uvicorn

Components:

  • FastAPI
  • SQLAlchemy
  • Uvicorn

REST Endpoints:

  • Web Pages
  • OpenAPI Docs
  • Info
  • Users
  • Weblist

To run in developer mode

# uvicorn webapp.main:app --reload

The budget objects are based on iCal formatted files so that they can easily be added to emails and calendars. Example:

iCAL 
          
BEGIN:VCALENDAR
VERSION:2.0
PRODID:-//joshroche
BEGIN:VEVENT
UID:20230330T162101CET@fd42:ad2:b23e:c209::1/64
TRANSP:TRANSPARENT
DTSTART:20230330T162101L
DTEND:20230330T162101L
DTSTAMP:20230330T162101L
ORGANIZER;CN=Josh Roche:mailto:josh_roche@hotmail.com
SUMMARY:$123.45
DESCRIPTION:
Electric Bill

LOCATION:https://www.paypal.com/
END:VEVENT
END:VCALENDAR
